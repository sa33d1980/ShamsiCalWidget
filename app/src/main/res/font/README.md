## فونت Vazirmatn

فایل‌های فونت زیر را از Google Fonts دانلود کنید و در این پوشه قرار دهید:

https://fonts.google.com/specimen/Vazirmatn

فایل‌های مورد نیاز:
- vazirmatn_medium.ttf  ← Vazirmatn-Medium.ttf را rename کنید
- vazirmatn_bold.ttf    ← Vazirmatn-Bold.ttf را rename کنید

سپس فایل res/font/fonts.xml باید وجود داشته باشد (در پروژه موجود است).
